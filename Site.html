Home Page

The home page implements the "Heritage Racing Sophisticate" aesthetic: a deep cold navy-black (#0D0F12) that makes car photography "sing," with surgical cyber-lime (#E2FF00) reserved exclusively for the primary CTA button. The hero uses a parallax effect where the title block shifts at 0.28× scroll speed, creating depth perception as the user scrolls. "DRASA" renders as solid white (foreground), "BUILDS." renders as an outline-only stroke (receding into the image, creating a layered depth illusion). The scroll indicator is positioned at the bottom-right, clear of all CTA buttons. Services use a Bento Grid layout with glass-panel cells (gradient borders via CSS mask technique). Buttons feature a "liquid sweep" hover animation using a skewed pseudo-element. The Syne typeface (editorial grotesque) paired with JetBrains Mono (technical/engineering data labels) signals both design taste and frontend expertise to hiring managers.

About DRASA Motorsports

The About page opens with a brand names strip — car makes displayed at 14% opacity, brightening to lime on hover — communicating DRASA's range without cluttering the layout. The sticky photo panel uses filter:brightness(.82), the minimum needed to allow the lime top-border to read. A full-width cinematic cinema strip closes the editorial section.

Gallery

The Gallery uses a Norris-style masonry card grid referencing the landonorris.com helmet gallery — rounded cards (3px), lime border on hover, subtle scale transform, build name left and category tag right at the bottom of each card (tag fades in on hover). Images use filter:saturate(.9) brightness(.88) — minimal degradation that maintains colour fidelity while providing consistent dark-panel integration.

Registration & Confirmation

Registration implements the motivation-before-action UX principle: the left panel shows a full-height build photo with benefit items (motivation), the right panel delivers the form (action). Labels are always visible above inputs (never placeholder-only). Gender tiles show a gradient-border glass selected state. The Confirmation page uses an animated lime square checkmark via CSS keyframes.

Feedback & Confirmation

Feedback uses a sidebar + content layout. Stars are keyboard-accessible with ARIA roles. Experience cards have clear selected states with gradient borders. The Feedback Confirmation uses a split: editorial text left with a scanning lime animation bar, Car.gif animation right — kinetic brand celebration appropriate to DRASA's speed-focused identity.
